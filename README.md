# Powerbi_Analises - Projetos e Dashboards

Repositório com projetos de análise de dados e dashboards interativos desenvolvidos no **Microsoft Power BI**, utilizando dados de exemplo para demonstrar conceitos e funcionalidades da ferramenta.

---

## 📂 Estrutura do Repositório

### 📊 Arquivos Power BI (.pbix)

| Arquivo | Descrição |
|---------|-----------|
| `Analise_Venda_Cafe.pbix` | Dashboard de análise de vendas de café, com métricas e visuais sobre desempenho de produtos e regiões. |
| `MOM and YOY.pbix` | Análise de variação **Mês a Mês (MoM)** e **Ano a Ano (YoY)** para acompanhamento de crescimento e tendências. |
| `Projeto_Orcamento.pbix` | Dashboard de acompanhamento orçamentário, comparando receitas, despesas e variações. |
| `Venda.pbix` | Análise geral de vendas, provavelmente com indicadores de performance (KPIs) e filtros interativos. |

---

### 📁 Bases de Dados

| Arquivo | Descrição |
|---------|-----------|
| `Base Vendas.xlsx` | Base de dados de vendas utilizada nos dashboards. |
| `Base_vendas.xlsx` | (Versão alternativa ou atualizada da base de vendas) |
| `Orcamento.xlsx` | Dados de orçamento, receitas e despesas para o projeto de orçamento. |
| `Coffe_sales.csv` | Arquivo CSV com dados de vendas de café (utilizado no dashboard de café). |

---

### 🧠 Pasta `Power BI DAX`

Contém scripts e expressões **DAX (Data Analysis Expressions)** utilizadas nos projetos, como:
- Cálculos de medidas
- Colunas calculadas
- Fórmulas de inteligência de tempo (MoM, YoY, etc.)
